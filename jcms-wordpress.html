Plugin JCMS Connector

Summary

A shortcode-based plugin for WordPress that allows embedding of remote text content from, for example, an existing content management system. Features include:

  • Up to three server connections
  • Simple shortcode embedding of CMS content
  • Configurable caching of CMS content

Plugin WP-LibriVox

Summary

A shortcode-based plugin for WordPress that accesses free public domain readings of out-of-copyright and copyright-free books. Recordings are provided by LibriVox.org and are accessed via the LibriVox REST API

  • Presents a single book with playable tracklist using it's LibriVox ID.
  • Provides a rich web application that allows direct searching, browsing and playing of any recording in the LibriVox catalogue
  • Plugin provides a proxy REST API to the LibriVox API. This proxy API is also available should other plugins wish to use it.
